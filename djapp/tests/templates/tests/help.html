ПОМОЩЬ

1. Создание нового пользователя

1.1. Для создания нового пользователя необходимо зайти в приложение от имени администратора (shorn)

1.2. После входа от имени администратора вы увидите страницу с результатами прохождения тестирования различными пользователями

1.3. Теперь необходимо перейти в админ панель для этого изменим адрес запроса с http://127.0.0.1:1337/administer на http://127.0.0.1:1337/admin, после чего откроется админ панель

1.4. В админ панеле переходим в раздел "Информация о тестируемых пользователях"

1.5. На открывшейся странице выбираем "ДОБАВИТЬ ИНФОРМАЦИЮ О ТЕСТИРУЕМОМ ПОЛЬЗОВАТЕЛЕ"

1.6. В открывшемся окне геобходимо заполнить информацию о тестируемом пользователе

1.7. Для этого нажимаем на " + "

1.8. И в открывшемся диологовом окне создаем пользователя задавая ему логин и придумывая пароль и вконце сохраняем результат

1.9. После возврата на предыдущую страницу заполняем остальные данные по пользователю так что бы результата выглядил приблизительно следующим образом

Обратите внимание что информация в поле "Список доступных для пользователя тестов и отметка о их прохождении:" приведена в JSON формате:


      [
        {
            "ended": true,
            "answer_id": 2,
            "name_test": "Тест Кеттелла"
        },
        {
            "ended": true,
            "answer_id": 2,
            "name_test": "Тест СМИЛ"
        },
        {
            "ended": true,
            "answer_id": 2,
            "name_test": "Тест УСК"
        },
        {
            "ended": true,
            "answer_id": 3,
            "name_test": "Тест для прогрессивных матриц Равена"
        },
        {
            "ended": true,
            "answer_id": 3,
            "name_test": "Тест для сложных аналогий"
        }
      ]

В разделе ended установлен флаг прохождения теста true - пройден, false - не пройден;

В разделе answer_id установлено число-ссылка на соответствующий идентификатор ответа по соответствующему тесту;

В разделе name_test установлен требуемый для прохождения тест (доступно пять вариантов: Тест Кеттелла, Тест СМИЛ, Тест УСК, Тест для прогрессивных матриц Равена, Тест для сложных аналогий)


Для установки начальных значений необходимо установить следующие:


      [
        {
            "ended": false,
            "answer_id": -1,
            "name_test": "Тест Кеттелла"
        },
        {
            "ended": false,
            "answer_id": -1,
            "name_test": "Тест СМИЛ"
        },
        {
            "ended": false,
            "answer_id": -1,
            "name_test": "Тест УСК"
        },
        {
            "ended": false,
            "answer_id": -1,
            "name_test": "Тест для прогрессивных матриц Равена"
        },
        {
            "ended": false,
            "answer_id": -1,
            "name_test": "Тест для сложных аналогий"
        }
      ]

После заполнения всех полей необходимо нажать на кнопку "СОХРАНИТЬ", после чего создание пользователя будет завершено.
